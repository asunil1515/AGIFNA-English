**Event Details:**
- July 23–26, 2026
- Houston Marriott Westchase, Houston, TX

## Features
- Responsive design (desktop → mobile)
- Countdown timer
- Scroll-triggered reveal animations
- Accordion FAQ
- Mobile navigation drawer
